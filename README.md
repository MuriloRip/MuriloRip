### 🚀 Projetos em Destaque

Nesta seção, destaco os repositórios que demonstram minhas habilidades técnicas em diferentes ecossistemas.

#### 📋 [TaskFlow API](https://github.com/MuriloRip/taskflow-api)
*REST API para gerenciamento de tarefas e projetos com Java Spring Boot.*
- **Tecnologias**: Java, Spring Boot, Spring Security, JWT, PostgreSQL, Docker.
- **Destaques**: Autenticação robusta, RBAC (Role-Based Access Control) e cobertura total de testes unitários.

#### 📚 [Bookshelf API](https://github.com/MuriloRip/bookshelf-api)
*Catálogo de livros e sistema de reviews em Node.js.*
- **Tecnologias**: TypeScript, Node.js, Express, Prisma ORM, PostgreSQL.
- **Destaques**: Validação de inputs rigorosa, paginação eficiente e design modular seguindo Clean Architecture.

#### 🔍 [LogSentry](https://github.com/MuriloRip/logsentry)
*Microserviço de análise de logs em tempo real.*
- **Tecnologias**: Python, FastAPI, Asyncio, Pydantic, SQLite.
- **Destaques**: Processamento assíncrono para ingestão de logs e painel de analytics estruturado para monitoramento.
